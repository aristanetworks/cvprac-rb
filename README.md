# Cvprac
Cvprac is a REST API Client for Arista CloudVision&reg; Portal (CVP).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'cvprac'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install cvprac
